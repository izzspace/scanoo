# Scanoo — Messages de prospection LinkedIn

## 🎯 Cibles prioritaires

### Secteur 1 : Restaurants / Hôtels / Bars
- Gérants, propriétaires, directeurs
- Facile à trouver sur LinkedIn
- Besoin évident (Google Maps, avis, site web)

### Secteur 2 : Artisans (plombiers, électriciens, maçons, couvreurs)
- Souvent un site web pourri ou inexistant
- Très dépendants du référencement local
- Budget serré = notre prix les séduit

### Secteur 3 : Professions libérales (avocats, kinés, dentistes, comptables)
- Ont un site mais ne l'optimisent jamais
- Image pro importante
- Pouvoir d'achat plus élevé

### Secteur 4 : Agents immobiliers
- Ultra concurrentiel, visibilité = survie
- Beaucoup sur LinkedIn

### Secteur 5 : Coachs, consultants, formateurs
- Dépendent de leur image en ligne
- Très actifs sur LinkedIn

## 🔑 Règles de prospection

### Volume
- **10-20 messages/jour max** (LinkedIn bloque au-delà)
- Espacer les envois (pas tout en 5 min)
- Varier les messages (pas de copié-collé exact)

### Personnalisation obligatoire
- Toujours mentionner le NOM du business
- Toujours mentionner la VILLE
- Toujours inclure UN PROBLÈME RÉEL trouvé sur leur site
- C'est ça qui fait la différence avec du spam

### Timing
- Envoyer entre 8h-11h et 14h-17h (heures de bureau)
- Mardi, mercredi, jeudi = meilleurs jours
- Éviter lundi matin et vendredi après-midi

### Process avant chaque message
1. Trouver le prospect sur LinkedIn
2. Lancer un mini-audit rapide de son site (node audit.js [url])
3. Identifier 1-2 problèmes concrets
4. Personnaliser le message avec ces infos
5. Envoyer

### Suivi
- Tracker chaque prospect dans un fichier (nom, business, messages envoyés, réponses)
- Message 1 → attendre 3 jours → Message 2 → attendre 4 jours → Message 3
- Après Message 3 sans réponse → on passe au suivant
